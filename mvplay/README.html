■mvplay v1.0

  • 分類： JavaScript で書いた動画再生ツール
  • 配布： https://bowlroll.net/file/319659

●おもなファイル

  • index.html // トップページ
  • mvplay.html // プレイヤー本体
  • movie_data.js // 動画登録のサンプル

●使い方

  • ブラウザで index.html を開くと、登録された動画を順次再生します
  • movie_data.js に、自分の見たい動画を登録してください
  • 登録形式は、"動画URL|ニコ動ID|動画タイトル" をカンマ区切りで並べます
  • 動画URL は、公開された動画ファイルの生のURLです
  • または、ローカルファイルの相対パス でもよいです
  • というか元々このツールはローカルPCの動画を見るために作ったものです

●動画品質の選択

  • 通信データ量を減らしたい人のための機能です
  • 異なる品質の動画を用意し、特定のフォルダに入れることで、品質を切り替えられます
  • [中品質] - mv60 フォルダの動画を再生します
  • [低品質] - mv30 フォルダの動画を再生します
  • [無品質] - mv15 フォルダの動画を再生します
  • たとえば https://bagof.crap.jp/coe/ みたいなフォルダ構成にします

●タイトル検索

  • キーワードを指定すると、それを含む動画タイトルを検索します
  • 「！キーワード」は、キーワードを含まない動画を検索します
  • 「キーワードＡ　＆　キーワードＢ」と書くと、AND 検索になります
  • 「キーワードＡ　｜　キーワードＢ」と書くと、OR 検索になります
  • たとえば「Ｃ子 ｜ Ｄ子 ＆ ！寸劇」は、
    "Ｃ子" または "Ｄ子" を含む、"寸劇" でない動画
    という意味になります（ANDとORの優先順位はなく、左から右に実行）
  • なお「＆」は「＞」でもよく、「｜」は「＜」でもよいです
  • 「＞」は絞り込む、「＜」は拡張する、みたいなイメージです
